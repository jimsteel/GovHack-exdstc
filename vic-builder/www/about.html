About

Victoria Builder is an online visual reference of building activity in Victoria. It exposes data previously stored in extremely large spreadsheets.

Attributions:

  • Street addresses via Google Geocoding API
  • Street photographs via Google StreetView
  • Graph Visualizations with Mike Bostock's D3
  • Database and website hosted on NeCTAR
  • Victorian Building Authority data provided by State Government of Victoria
